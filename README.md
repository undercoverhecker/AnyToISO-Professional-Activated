## üé• How to Use AnyToISO
### 1. **Creating an ISO from Files**
- Open AnyToISO.
- Select "File to ISO".
- Browse and select the files or folders you want to convert.
- Click "Convert" to create the ISO file.

### 2. **Creating an ISO from a CD/DVD**
- Insert the disc into your CD/DVD drive.
- Open AnyToISO.
- Select "CD/DVD to ISO".
- Click "Extract" to save the ISO file to your computer.

### 3. **Extracting Files from an ISO**
- Open AnyToISO.
- Select "ISO to Files".
- Browse and choose the ISO file you want to extract.
- Click "Extract" to save the files to your preferred location.
